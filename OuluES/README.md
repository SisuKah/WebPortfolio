## 🚀 Oulu Entrepreneurship Society Website Platform

This project is the **official website platform** for **Oulu Entrepreneurship Society (OuluES)**, built to grow a student startup community in Oulu. 🌍🔥

It presents the organization’s core message — **“for the ones who build”** — while serving as both a **brand home** and a **community onboarding channel** for future founders.

The platform includes:

- 🏠 **Modern landing page**
- 📅 **Event promotion pages**
- 👥 **Board member introductions**
- 📚 **Startup activity directory**
- 🌱 **Community growth & founder onboarding focus**

This helps position OuluES as the central hub for entrepreneurship and startup culture in Oulu.

---

## ⚙️ Platform Features

Technically, it is a modern **Next.js + React** web application with **Firebase-backed authentication and data support**. ⚡

It also includes an internal **admin experience** for maintaining content and custom pages without hardcoded edits.

Key functionality includes:

- 🔐 Admin authentication
- ✍️ Content management tools
- 📄 Custom page editing
- 👤 Board member management
- 🔄 Dynamic content updates
- 🌐 API routes for internal operations

---

## 🛠️ Tech Stack

Built using:

- ▲ **Next.js**
- ⚛️ **React**
- 🔥 **Firebase**
- 🟦 **TypeScript**

This makes the platform scalable, maintainable, and fast for everyday use.

---

## 🎯 Final Result

Rather than being a one-time static website, this is a **living digital platform** designed for the ongoing needs of the organization. It supports branding, community growth, and day-to-day operations while helping inspire the next generation of founders in Oulu. 🚀
