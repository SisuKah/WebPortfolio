## 🔐 SmartLock Company Website

This project is a modern website built for a Finnish smart lock company, created to present products clearly and support conversion with strong call-to-actions.

The site emphasizes responsive design and practical product presentation across devices.

---

## ⚙️ Platform Highlights

The website includes:

- 📱 Responsive, mobile-friendly layout
- 🧱 Product showcase sections
- 🎯 Clear call-to-actions for promotion and sales

---

## 🛠️ Tech Stack

Built using:

- ▲ **Next.js**
- 🟦 **TypeScript**
- 🎨 **Tailwind CSS**

---

## 🎯 Final Result

The end result is a clean and business-focused product website that helps communicate value quickly while supporting smart lock marketing and sales goals. ✅
