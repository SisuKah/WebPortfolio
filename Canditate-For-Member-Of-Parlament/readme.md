## 🗳️ Political Campaign Website – Raimo Rautiola (Vapauden Liitto)

This project was a **personal political campaign website** built for Raimo Rautiola (Vapauden Liitto), designed to present his **profile, values, and current writings** in a clean, modern, and mobile-friendly format. 📱✨

The public site includes:

- 🏠 **Homepage**
- 📰 **Post listing page**
- 📄 **Individual article views**
- 🔗 **Sharing features**
- 🌐 **URL-based deep linking** (`?post=`)
- 🚀 **SEO & social metadata** *(Open Graph + Twitter Cards)* for easier discovery and sharing

It also supports **graceful fallback content** from local static posts if live data is unavailable, helping keep the website functional during setup or outages. 🛡️⚡

---

## ⚙️ Custom Admin System

A major part of the implementation was a **custom content management workflow**, including:

- 🔐 Authenticated login
- ✍️ Create / Edit / Delete blog posts
- 🖼️ Image uploads to storage
- 📌 Post pinning for prioritizing key campaign messages

---

## 🛠️ Tech Stack

Content is managed through **Supabase** for:

- 🗄️ Database
- 🔑 Authentication
- ☁️ Storage

The frontend was built using:

- ⚛️ React
- 🟦 TypeScript
- ⚡ Vite

This made the project **fast to develop, scalable, and easy to maintain**.

---

## 🎯 Final Result

Overall, this is a **practical full-stack publishing platform** tailored for political communications, campaign messaging, and ongoing content updates. It gives the campaign a professional digital presence while making publishing simple and efficient. 🚀
