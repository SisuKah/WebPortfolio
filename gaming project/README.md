## 🧰 Minecraft Armor Durability Resource Pack Automation

This project focused on automating a Minecraft resource pack workflow that would have been extremely repetitive by hand.

I built Python scripts that generated **4000+ textures** and matching `.properties` files, turning manual production work into a repeatable pipeline. ⚙️

---

## ⚙️ What Was Automated

The scripting flow handled:

- 🎨 Gradient overlay generation
- 🔢 Durability number rendering on textures
- 📁 Automatic file structuring and output formatting
- 🧠 Bulk generation for large asset sets

The implementation used **Python** and **Pillow** to process and export pack assets efficiently.

---

## 📈 Outcome & Learnings

The project reached **1500+ downloads** from players.

It also taught an important personal lesson: even with harsh community feedback, creating useful tools and handling criticism constructively is part of long-term growth as a builder. 💪
