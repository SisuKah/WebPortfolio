## 🕊️ Qr-Memo Website Platform

This project is a modern website built for **Qr-Memo**, a company providing QR code–based digital memorial services.

Because the topic is sensitive, the design direction was intentionally more conservative, with a focus on clarity, trust, and accessibility.

---

## ⚙️ Platform Focus

The website includes:

- 🏠 A clean and responsive public-facing experience
- 💬 Clear communication of a sensitive service concept
- 🔐 Backend support for data, authentication, and image handling

---

## 🛠️ Tech Stack

Built using:

- ▲ **Next.js**
- ⚛️ **React**
- 🟦 **TypeScript**
- 🎨 **Tailwind CSS**
- 🔥 **Firebase** *(database, authentication, and image storage)*
- ▲ **Vercel** *(deployment)*

---

## 🎯 Final Result

The result is a professional service website that balances modern implementation with respectful UX choices for a delicate subject area. ✅
