## 📦 Storage Rental Web Platform

This project is a full-stack web application for renting small storage units online, built with a modern Next.js + TypeScript architecture.

It is designed to give customers a clear and smooth rental flow from browsing available units to checkout confirmation.

---

## ⚙️ Core Features

The platform includes:

- 🗺️ Interactive SVG floor plan for visual unit selection
- ✅ Real-time availability visualization
- 💳 Stripe-powered checkout with flexible durations and deposits
- 🔐 Firebase authentication
- 🗄️ Firestore-based availability management
- 📱 Responsive Tailwind CSS UI

---

## 🛠️ Tech Stack

Built using:

- ▲ **Next.js**
- ⚛️ **React**
- 🟦 **TypeScript**
- 🎨 **Tailwind CSS**
- 💳 **Stripe**
- 🔥 **Firebase / Firestore**

---

## 🎯 Final Result

The final platform provides an end-to-end storage rental experience that is both customer-friendly and operationally practical, combining visual booking, secure payments, and manageable backend data flow. 🚀
